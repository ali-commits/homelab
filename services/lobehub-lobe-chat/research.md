# Lobe Chat Research

## Description
Lobe Chat is an open-source, modern-design AI chat framework and a personal LLM productivity tool. It provides a user experience for various AI models like ChatGPT, OLLaMA, Gemini, and Claude.

## Main Features
- Open-source, modern design
- Supports multiple AI providers (OpenAI, Claude 3, Gemini, Ollama, DeepSeek, Qwen)
- Knowledge Base (file upload, knowledge management, RAG)
- Multi-Modals (Plugins/Artifacts), Thinking
- Speech-synthesis, multi-modal capabilities, plugin system, Agent Market
