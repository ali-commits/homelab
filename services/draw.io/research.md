# Draw.io Research

## Description
Draw.io (also known as diagrams.net) is free online diagram software and a JavaScript, client-side editor for general diagramming. It can be used as a flowchart maker, network diagram software, UML tool, ER diagram tool, etc. It also has a downloadable desktop application for offline use.

## Main Features
- Flowchart maker, network diagram software, UML online, ER diagram tool, database schema design, BPMN online, circuit diagram maker
- Extensive template and shape libraries
- Generate diagrams from text (PlantUML, Mermaid, CSV, SQL)
- Customizable editor
- Real-time collaboration with shared cursors
- Integrates with cloud storage
- Desktop application for offline use
