# DrawDB Research

## Description
DrawDB is a free, simple, and intuitive online database diagram editor and SQL generator. It is a web-based database entity relationship (DBER) editor and data modeler.

## Main Features
- Online database diagram editor
- SQL generator
- Visualize database schema, design database diagrams
- Export SQL scripts
- Customize editor
- Reverse engineer DDL scripts
