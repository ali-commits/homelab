# Dozzle Research

## Description
Dozzle is a lightweight, web-based, realtime log viewer specifically designed for Docker containers. It provides an easy way to view logs in the browser, making debugging and troubleshooting easier.

## Main Features
- Realtime log streaming
- Filtering, searching
- Supports JSON logs with color coding
- Easy to install and configure
- Supports Docker Swarm mode
- Connect to remote hosts securely with agents
- Analyze logs with SQL queries (using WebAssembly and DuckDB)
