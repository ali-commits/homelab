# AFFiNE Research

## Description
AFFiNE is a next-gen knowledge base that aims to bring planning, sorting, and creating together. It is a workspace with fully merged docs, whiteboards, and databases, designed to be privacy-first, open-source, customizable, and ready to use. It is presented as an alternative to Notion and Miro.

## Main Features
- Knowledge base, planning, sorting, creating
- Docs, whiteboards, databases
- Block-based editor (convert any block with one click)
- Collaborative
- Privacy-first, open-source, customizable
