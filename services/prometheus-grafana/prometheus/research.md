# Prometheus Research

## Description
Prometheus is an open-source systems monitoring and alerting toolkit and time series database, originally built at SoundCloud. It is used for event monitoring and alerting, recording metrics in a time series database.

## Main Features
- Multi-dimensional data model
- Concise and powerful query language (PromQL)
- Efficient embedded time series database
- Collects metrics from targets via HTTP pull model
- Evaluates rule expressions, displays results, triggers alerts
- Client libraries for easy instrumentation
- Exporters for third-party data integration
