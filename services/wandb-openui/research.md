# OpenUI Research

## Description
OpenUI is a software that lets you describe UI using your imagination and then see it rendered live. It is part of the Weights & Biases (wandb) project, which is an AI developer platform.

## Main Features
- Describe UI with imagination, live UI rendering
- Convert HTML to various frameworks (React, Svelte, Web Components, etc.)
- Supports various LLMs via LiteLLM
- Related to AI development and Weights & Biases platform
