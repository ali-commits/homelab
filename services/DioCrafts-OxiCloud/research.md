# OxiCloud Research

## Description
OxiCloud is a lightweight, Rust-based, self-hosted file storage solution and server, presented as an efficient and secure way to save your data and an alternative to Nextcloud and Dropbox.

## Main Features
- Self-hosted file storage
- Lightweight, efficient, secure
- Rust-based
- Alternative to Nextcloud/Dropbox
