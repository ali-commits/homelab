# ChartDB Research

## Description
ChartDB is a free and open-source, web-based database diagramming editor that allows you to visualize and design your database with a single query. It helps you draw DB relationship diagrams and export DDL scripts.

## Main Features
- Web-based database diagramming editor
- Visualize database schema with a single query ("Smart Query")
- Customize diagrams, export SQL scripts
- Draw DB relationship diagrams, export DDL scripts
- AI for DDL script generation
- User-friendly
