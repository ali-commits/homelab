# Hoarder (Karakeep) Research

## Description
Karakeep (previously Hoarder) is an open-source, self-hostable "Bookmark Everything" app that uses AI for automatically tagging content. It allows you to bookmark links, take simple notes, and store images and PDFs.

## Main Features
- Bookmark links, notes, images, PDFs
- AI-based automatic tagging
- Automatic fetching for link titles, descriptions, and images
- Full-text search
- Chrome plugin
- iOS app
- Dark mode
- Content archiving (planned)
