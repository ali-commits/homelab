# Scrutiny Research

## Description
Scrutiny is a Hard Drive Health Dashboard & Monitoring solution that merges manufacturer-provided S.M.A.R.T metrics with real-world failure rates. It provides a web UI to show statistics on drives and report their health.

## Main Features
- Hard Drive S.M.A.R.T Monitoring
- Historical Trends
- Real World Failure Thresholds
- Web UI
- Health Dashboard
