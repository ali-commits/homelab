# Stirling-PDF Research

## Description
Stirling-PDF is a robust, locally hosted web-based PDF manipulation tool that allows you to perform various operations on PDF files. It is designed to be personalized and does not initiate any outbound calls.

## Main Features
- Splitting, merging, converting, reorganizing, adding images, rotating, compressing PDFs
- Cleanup scans
- Detecting text from images
- Chapter-based splitting
