# Joplin Research (Server Component)

## Description
Joplin is a free and open-source desktop and mobile note-taking and to-do list application with sync capabilities. The Joplin Server is a component that allows for self-hosted synchronization of notes across devices.

## Main Features (Joplin Application)
- Note taking, to-do lists
- Sync capabilities (multiple methods supported)
- Encryption
- Searchable notes, organize into notebooks, tagging
- Modify notes with external text editor
- Web clipper extension, save web pages/screenshots
- Attach files (PDFs, images), OCR for searching within attachments
- Create math expressions, diagrams
- Share notes and collaborate (with Joplin Cloud/Server)
- Publish notes to the internet
