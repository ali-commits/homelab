# Docmost Research

## Description
Docmost is an open-source collaborative wiki and documentation software, presented as an alternative to Confluence and Notion.

## Main Features
- Collaborative wiki, documentation software
- Real-time editor (multiple users simultaneously)
- Support for Spaces with permissions
- Rich text editor with markdown shortcuts
- Import and export pages (Markdown, HTML)
- Available in multiple languages
