# Checkmate Research

## Description
Checkmate is an open-source, self-hosted tool designed to track and monitor server hardware, uptime, response times, and incidents in real-time with beautiful visualizations. It was formerly known as BlueWave Uptime and consists of a frontend and a backend.

## Main Features
- Track and monitor server hardware
- Uptime monitoring
- Response time monitoring
- Incident tracking
- Real-time monitoring
- Beautiful visualizations
- Open-source, self-hosted
