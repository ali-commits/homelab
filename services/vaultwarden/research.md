# Vaultwarden Research

## Description
Vaultwarden is an unofficial, community-driven, Bitwarden-compatible server implementation written in Rust. It's designed for self-hosted deployments and is compatible with official Bitwarden clients. It was formerly known as bitwarden_rs.

## Main Features
- Bitwarden API compatibility (including organizations, attachments, vault API, website icons API, authenticator, U2F)
- Lightweight
- Suitable for self-hosting on resource-constrained devices
- Send feature (for encrypted file sharing)
