# Paperless-ngx Research

## Description
Paperless-ngx is a community-supported, supercharged version of Paperless that allows you to scan, index, and archive physical documents. It provides a web UI for filtering, viewing, searching, and editing documents, and managing tags, correspondents, and document types.

## Main Features
- Scan, index, and archive documents
- Web UI for management
- Filtering, viewing, searching, and editing documents
- Managing tags, correspondents, and document types
- OCR processing
- Support for various document types (with external tools)
