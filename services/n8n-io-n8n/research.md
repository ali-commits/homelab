# n8n Research

## Description
n8n is a free and source-available workflow automation platform with native AI capabilities. It allows you to combine visual building with custom code and integrate with over 400 services. It can be self-hosted or used in the cloud.

## Main Features
- Workflow automation, visual workflow building
- Custom code integration
- Native AI capabilities
- 400+ integrations/connectors
- Self-hostable, cloud option
- Rapid application development
- Easy-to-understand structure, save entire processes
- Build smart context-aware LLM applications
