# Firefly III Research

## Description
Firefly III is a free and open-source personal finance manager designed for self-hosting. It uses a double-entry bookkeeping system and helps you keep track of your expenses and income.

## Main Features
- Double-entry bookkeeping
- Track expenses and income
- Organize transactions in multiple currencies
- Data Importer for importing data
- Budgeting, categories, tags
- Customizable rule-engine
- Charts and visualizations
